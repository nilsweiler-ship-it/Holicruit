# Outreach email variants by segment

Reusable, pre-personalized versions of the two base emails. Each keeps the same
core value prop but leads with the pain that segment feels most. Fill the
`{curly}` blanks. Base versions live in `email-employers.md` and
`email-training-providers.md`.
